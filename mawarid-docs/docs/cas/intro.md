# CAS — Candidate Acceptance System

The **CAS (Candidate Acceptance System)** is used to manage and track the acceptance process for recruitment candidates.

## Overview

CAS handles the candidate acceptance workflow, allowing recruiters and managers to review, approve, or reject candidates at various stages of the hiring process.

## Key Features

- Candidate profile review
- Acceptance / rejection workflow
- Stage-based tracking of candidate status
- Notifications to relevant stakeholders
