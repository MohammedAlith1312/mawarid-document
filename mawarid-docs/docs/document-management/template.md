# Template

The **Template** page allows users to create and manage reusable document request templates. Templates save time by pre-filling approver setups and document configurations.

## Functions

- Create a request template from an existing document type
- Define approver positions and signature locations in the template
- Reuse templates when creating new requests to apply the same configuration

## Benefits

- Eliminates repetitive setup for similar document types
- Ensures consistency across requests of the same type
- Templates can be shared across users based on role access

## Notes

- Templates do not contain document files — only configuration and approver chain setup.
- Editing a template does not affect previously submitted requests that used it.
