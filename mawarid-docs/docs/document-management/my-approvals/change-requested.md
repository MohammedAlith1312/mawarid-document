# My Approvals — Change Requested

The **My Approvals - Change Requested** page shows requests where you have previously sent a **Change Request** back to the requester and are now waiting for the requester to resubmit.

## Functions

- View all requests you returned for changes
- Track whether the requester has resubmitted

## Notes

- Once the requester resubmits, the request moves back to your **Pending Approvals**.
- Your original change request comments remain visible in the Request Details.
