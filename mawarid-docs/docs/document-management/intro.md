# Document Management

The **Mawarid Document Management** portal is a fully static application for managing digitally signed documents between requesters and approvers.

## How It Works

1. **Users upload documents** and send them for approval with position-based approver assignments.
2. **Approvers review** the document and can: **Approve**, **Request a Change**, or **Reject**.
3. If approved, the approver's **digital signature and stamp are automatically uploaded** onto the document.

> Approver signatures are configured per position — the system places signatures based on the approver's designated position setup.
