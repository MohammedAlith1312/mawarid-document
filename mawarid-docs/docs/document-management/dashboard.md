# Dashboard

The **Dashboard** is the main landing page of the Document Management portal. It gives the user a quick summary of their request and approval activity.

## Functions

- Displays **count** of **My Requests** grouped by status
- Displays **count** of **My Approvals** grouped by status

## Counters Shown

| Counter | Description |
|---|---|
| Draft | Requests in draft stage |
| In Review | Requests currently under review |
| Change Requested | Requests returned for changes |
| Approved | Requests that have been approved |
| Rejected | Requests that have been rejected |
| Pending Approvals | Requests awaiting your approval action |
