# Parameter

The **Parameter** page allows administrators to configure system-wide default settings for the Document Management application.

## Functions

- Set and manage system-level default parameters
- Configure application behavior and defaults

## Common Parameters

| Parameter | Description |
|---|---|
| Default Sign Type | Default signature type applied to new approvers |
| Max Document Size | Maximum allowed file upload size |
| Notification Settings | Email/alert notification defaults |

## Notes

- Only admin users have access to this page.
- Changes to parameters take effect immediately for new requests.
