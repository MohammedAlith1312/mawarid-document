# Users

The **Users** page allows administrators to view and manage all users registered in the Document Management application.

## Functions

- View the full list of registered users
- Search and filter users by name or role
- Manage user access and role assignments

## Notes

- Only admin users have access to this page.
- User accounts are linked to their approver signature setup in the [Approver](./approver) page.
