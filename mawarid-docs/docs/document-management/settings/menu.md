# Menu

The **Menu** page allows administrators to manage the application menu structure and assign menu items to roles.

## Functions

- View the full menu list of the Document Management application
- Assign menu items to specific **roles** (role-based menu access)

## How It Works

- Each menu item can be restricted to one or more roles.
- Users will only see menu items that their assigned role has access to.

## Notes

- Only admin users have access to this page.
- Changes to menu role assignments take effect on the user's next login.
