# Roles

The **Roles** page allows administrators to manage application roles and assign users to those roles.

## Functions

- View the full list of available roles
- Create and configure roles with specific permissions
- Assign users to roles

## Available Roles

| Role | Access Level |
|---|---|
| Admin | Full access to all pages and settings |
| Requester | Can create and submit document requests |
| Approver | Can review and approve/reject/change-request documents |
| Viewer | Read-only access to approved documents |

## Notes

- A user can be assigned multiple roles.
- Role assignments control both menu visibility (see [Menu](./menu)) and action permissions.
- Changes take effect immediately on the user's next session.
