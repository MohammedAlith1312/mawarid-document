# My Request — Draft

The **My Request - Draft** page shows all document requests you have created that are still in **Draft** status and have not been submitted yet.

## Functions

- View all your draft requests
- **Submit** the draft to send it for approval
- **Add Approvals** — configure the approver list, sign type, and signature location before submitting

## Actions Available

| Action | Description |
|---|---|
| Submit | Sends the request to approvers — transitions status to **In Review** |
| Add Approvals | Add approvers with their sign type and position on the document |
| Edit | Modify document details before submission |
| Delete | Delete the draft permanently |

## Notes

- Draft requests are only visible to the requester.
- A request must have at least one approver added before it can be submitted.
