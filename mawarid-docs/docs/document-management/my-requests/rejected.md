# My Request — Rejected

The **My Request - Rejected** page shows all your document requests that have been **rejected** by an approver.

## Functions

- View all your rejected requests
- Review the rejection reason provided by the approver

## Notes

- Rejected requests cannot be resubmitted directly.
- A new request must be created if the document needs to go through the process again.
- Rejection reason and approver comments are visible in the [Request Details](./request-details) page.
