# Request Details

The **Request Details** page provides the full detail view of a document request, including the approver chain, document history, and status timeline.

## Functions

### Request Details & Actions
- View complete request information
- Perform available actions based on current status (submit, resubmit, etc.)

### Approvers List
- View all approvers assigned to this request
- Each approver entry shows:
  - Approver name and position
  - **Sign Type** (e.g., signature, stamp, or both)
  - **Signature location** on the document
  - Current approval status

### Document History
- Shows a chronological list of all document file versions
- Each entry represents a change or upload to the document file
- Useful for tracking what changed between versions

### Status History
- Full timeline of all status transitions for the request
- Shows who performed each action and when

## Notes

- This page is accessible from any request list (Draft, In Review, Approved, etc.) by clicking on a request.
