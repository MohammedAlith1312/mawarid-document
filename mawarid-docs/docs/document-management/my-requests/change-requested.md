# My Request — Change Requested

The **My Request - Change Requested** page shows requests that have been returned by an approver with a request for changes.

## Functions

- View all your requests that require changes
- Review approver comments and feedback
- Resubmit after making the required changes

## Workflow

```
In Review → Change Requested → [User edits] → In Review
```

## Notes

- The requester must address the approver's comments and resubmit.
- Only the fields permitted by the approver's change request can be modified.
- Resubmitting returns the document to **In Review** status.
