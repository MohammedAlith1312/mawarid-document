# Work Order Task

The **Work Order Task** page manages the individual tasks that belong to a work order.

## Functions

- View and manage tasks within a work order
- Assign tasks to specific employees
- Track task-level completion status
- Record materials used and time spent

## Task Status

| Status | Description |
|---|---|
| Pending | Task not yet started |
| In Progress | Task is being worked on |
| Completed | Task finished |

## Notes

- Each work order can have multiple tasks.
- Tasks are linked to a Work Activity and may use defined Materials.
- Employee assignments at the task level are reflected in [My Task](../my-task/pending).
