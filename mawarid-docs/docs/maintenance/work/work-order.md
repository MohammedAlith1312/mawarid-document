# Work Order

The **Work Order** page manages maintenance work orders generated from work plans or created manually.

## Functions

- Create, view, and manage work orders
- Assign work orders to sites, locations, assets, and employees
- Track work order status

## Work Order Status

| Status | Description |
|---|---|
| Pending | Work order created, not yet started |
| In Progress | Work is currently being carried out |
| Completed | All tasks have been completed |
| Cancelled | Work order cancelled before completion |

## Notes

- Work orders contain one or more [Work Order Tasks](./work-order-task).
- Work orders can be created manually or auto-generated from a Work Plan.
