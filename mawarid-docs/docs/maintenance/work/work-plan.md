# Work Plan

The **Work Plan** page is used to create and manage maintenance work plans that define scheduled maintenance activities.

## Functions

- Create and manage maintenance work plans
- Define scope, schedule, and resources for planned maintenance
- Link work plans to assets, sites, and locations

## Notes

- Work plans are the top-level planning object — Work Orders are generated from plans.
- Plans can be set as recurring based on the schedule configuration.
