# My Site

The **My Site** page shows all maintenance sites assigned to the currently logged-in user.

## Functions

- View all sites the user is responsible for
- Access work orders and tasks specific to each site
- Monitor site-level maintenance status

## Notes

- Site assignments are configured in the [Site](./master/site) master data page.
- Only sites assigned to the user's role or account are shown.
