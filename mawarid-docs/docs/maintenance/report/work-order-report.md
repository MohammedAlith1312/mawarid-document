# Work Order Report

The **Work Order Report** provides a summary report of all work orders across sites and time periods.

## Functions

- Generate work order reports filtered by date range, site, status, or assignee
- View counts and status breakdown of work orders
- Export report data

## Report Columns

| Column | Description |
|---|---|
| Work Order No. | Unique identifier |
| Site | Site where the work was performed |
| Status | Current status of the work order |
| Created Date | Date the work order was created |
| Completed Date | Date the work order was completed |
| Assigned To | Employee or team responsible |
