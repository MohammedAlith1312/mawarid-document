# Work Order Task Report

The **Work Order Task Report** provides a detailed report of all tasks within work orders.

## Functions

- View task-level details across all work orders
- Filter by date range, site, work order, status, or work activity
- Export report data

## Report Columns

| Column | Description |
|---|---|
| Task No. | Unique task identifier |
| Work Order | Parent work order reference |
| Activity | Work activity type performed |
| Status | Task status |
| Asset | Asset the task was performed on |
| Location | Location where the task was done |
| Assigned To | Employee responsible |
