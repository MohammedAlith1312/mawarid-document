# Work Order Task Report By Employee

The **Work Order Task Report By Employee** provides a task report grouped and filtered by employee, showing each technician's workload and completion history.

## Functions

- View task reports filtered by employee
- Assess individual workload and performance
- Filter by date range, employee, site, or task status
- Export report data

## Report Columns

| Column | Description |
|---|---|
| Employee | Technician name |
| Task No. | Task identifier |
| Work Order | Parent work order reference |
| Activity | Work activity performed |
| Status | Task status |
| Site | Site where the task was performed |
| Completed Date | Date task was completed |
