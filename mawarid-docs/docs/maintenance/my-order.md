# My Order

The **My Order** page shows all work orders assigned to the currently logged-in user.

## Functions

- View all work orders assigned to the current user
- Track work order status (Pending, In Progress, Completed)
- Access tasks within each work order

## Notes

- Work orders are created and assigned via the [Work Order](./work/work-order) page.
- Users can drill down into each work order to view its tasks.
