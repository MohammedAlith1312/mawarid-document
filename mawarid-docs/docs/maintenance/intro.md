# Maintenance

The **Mawarid Maintenance Portal** provides tools to manage work plans, work orders, tasks, assets, and maintenance operations across sites.

## Sections

| Section | Description |
|---|---|
| [Dashboard](./dashboard) | Overview of maintenance activity |
| [Work](./work/work-plan) | Work Plans, Work Orders, and Work Order Tasks |
| [Master](./master/site) | Site, Location, Asset, and Employee master data |
| [My Site](./my-site) | Sites assigned to the current user |
| [My Order](./my-order) | Work orders assigned to the current user |
| [My Task](./my-task/pending) | Tasks assigned to the current user (Pending, Completed, All) |
| [Report](./report/work-order-report) | Work Order and Task reports |
| [Setup](./setup/work-category) | Configuration: Categories, Activities, Asset Types, Schedules, Materials, Location Types |
| [Security](./security/user-list) | User access management and login reports |
