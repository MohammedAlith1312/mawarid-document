# User Login Report

The **User Login Report** provides a history of user login activity in the Maintenance portal.

## Functions

- View login history for all users
- Filter by user, date range, or login status
- Monitor access patterns and detect anomalies

## Report Columns

| Column | Description |
|---|---|
| Username | The user who logged in |
| Login Date/Time | When the login occurred |
| IP Address | Source IP of the login |
| Status | Success or failed login attempt |
