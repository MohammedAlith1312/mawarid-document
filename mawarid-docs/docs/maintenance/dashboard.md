# Dashboard

The **Dashboard** provides an overview of maintenance activity across all sites, orders, and tasks.

## Functions

- Summary counts of active Work Orders and Tasks
- Status breakdown: Pending, In Progress, Completed
- Quick links to My Orders and My Tasks

## Displayed Widgets

| Widget | Description |
|---|---|
| Work Orders | Total active work orders summary |
| Tasks | Pending vs completed task counts |
| My Site Summary | Overview of sites assigned to the current user |
