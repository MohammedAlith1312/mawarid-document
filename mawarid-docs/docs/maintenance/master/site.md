# Site

The **Site** master page is used to define and manage maintenance sites.

## Functions

- Create and manage site records
- Assign sites to users and teams
- Sites serve as the top-level grouping for Locations and Assets

## Notes

- Sites appear in [My Site](../my-site) for assigned users.
- Each site contains one or more [Locations](./location).
