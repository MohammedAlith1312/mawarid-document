# My Pending Task

The **My Pending Task** page shows all maintenance tasks assigned to the current user that have not yet been completed.

## Functions

- View all tasks currently awaiting action
- Update task status (mark as In Progress or Completed)
- View task details: site, location, asset, activity, and materials

## Notes

- Tasks are assigned from [Work Order Task](../work/work-order-task).
- Completing a task here updates the parent Work Order progress.
