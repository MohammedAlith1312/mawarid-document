# My All Task

The **My All Task** page shows the complete list of all tasks assigned to the current user regardless of status.

## Functions

- View all tasks in one combined list (pending, in progress, and completed)
- Filter by status, date, site, or work order

## Notes

- Combines the data from [Pending](./pending) and [Completed](./completed) views.
- Useful for a full personal task history overview.
