# All Requests

The **All Requests** page provides admin-level access to view all requests across all employees and all request types.

## Page Type

- **Type:** Admin Access

## Requests Shown

| Request Type | Description |
|---|---|
| All Leave Requests | View all employee leave requests |
| All Loan Requests | View all employee loan requests |
| All Business Trips | View all business trip requests |
| All Purchase Requests | View all purchase requests |
| All Store Item Requests | View all store item requests |

## Filters Available

- Filter by employee name or ID
- Filter by request type
- Filter by workflow status (Draft, Submitted, Approved, Rejected, Error)
- Filter by date range

## Notes

- This page is restricted to **Admin** users only.
- Admins can view but not modify requests that are already in workflow.
