# My Approval

The **My Approval** page shows all requests pending the current user's approval action. It filters requests where the logged-in user is an assigned approver.

## Page Type

- **Type:** Same request filter by approvers

## Requests Shown

| Request Type | Filter |
|---|---|
| Leave Request | Same request filtered by approvers |
| Loan Request | Same request filtered by approvers |
| Business Trip | Same request filtered by approvers |
| Purchase Request | Same request filtered by approvers |
| Store Item Request | Same request filtered by approvers |

## Actions Available

- **Approve** — Move the request to the approved stage
- **Reject** — Move the request to the rejected stage
- **Cancel** — Cancel the request

## Notes

- Only requests assigned to the current user for approval are shown.
- The approver does not need to re-enter any data — they review and act on the submitted form.
