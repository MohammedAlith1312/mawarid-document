# My Request

The **My Request** page shows all requests submitted by the currently logged-in employee. It filters requests based on the current employee's identity.

## Page Type

- **Type:** Workflow filter by current employee

## Requests Shown

| Request Type | Filter |
|---|---|
| Leave Request | Filtered by current employee |
| Loan Request | Filtered by current employee |
| Business Trip | Filtered by current employee |
| Store Item Request | Filtered by current employee |

## Actions Available

- **View** — View the full request details
- **Cancel/Delete** — Cancel or delete a draft request
- **Track Status** — See the current workflow stage

## Notes

- Only requests created by the logged-in employee are displayed.
- Approved or submitted requests cannot be edited.
