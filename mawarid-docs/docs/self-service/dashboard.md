# Dashboard

The **Dashboard** is the main landing page of the Self Service portal. It provides a quick overview and access to all modules.

## Page Type

- **Type:** Custom Template

## Features

- Summary widgets for pending requests
- Quick links to submit new requests
- Notification center for approvals
