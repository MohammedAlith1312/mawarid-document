# My Location

The **My Location** page shows the location(s) that have been assigned to the logged-in provider by the Admin.

## Functions

- View the location(s) assigned to this provider account
- See location details (name, address, number of seats/cabins)

## Notes

- Providers can only see and serve locations they have been **assigned to** by the Admin.
- If no location is assigned, the provider will not receive any orders.
- Assignment is managed by the Admin from the [Assignment](../admin/assignment) page.
