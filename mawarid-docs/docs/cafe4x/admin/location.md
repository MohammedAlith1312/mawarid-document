# Location

The **Location** page allows administrators to manage site locations. A location represents a physical site (e.g. a work site or building) where cafeteria service is provided.

## Functions

- View all locations
- Create a new location
- Edit location details (name, description)
- Activate / Deactivate a location

## Sub-sections

Each location has two sub-sections managed from the sidebar:

| Sub-section | Description |
|---|---|
| [Location Seats](./location-seats) | Manage cabins/seats within the location and generate QR codes |
| [Assignment](./assignment) | Assign which providers serve this location |
