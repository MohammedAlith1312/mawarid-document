# Location Seats

The **Location Seats** page manages the individual seats (cabins) within a location. Each seat has a unique **QR code** that requesters scan to place their orders.

## Functions

- View all seats for a selected location
- Create a new seat (cabin)
- Edit seat details (seat number, label)
- **Generate QR Code** — generates a unique QR code for the seat
- Download / Print the QR code for physical placement at the seat
- Activate / Deactivate a seat

## How QR Codes Work

1. Admin creates a seat and generates its QR code.
2. The QR code is printed and placed at the physical seat/cabin.
3. A **Requester** scans the QR code from their seat.
4. The system automatically identifies the **location** and **seat** from the QR code.
5. The requester selects items and submits the order — no login required.
