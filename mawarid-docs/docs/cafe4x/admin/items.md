# Items

The **Items** page allows administrators to manage the cafeteria's product catalogue — the list of items available for requesters to order.

## Functions

- View all available items
- Create a new item (name, description, category, price)
- Edit item details
- Activate / Deactivate an item (inactive items are not shown to requesters)
- Upload item image

## Notes

- Only **active** items will be visible to requesters on the ordering screen.
- Items are shared across all locations.
