# Users

The **Users** page allows administrators to manage all user accounts in the Cafe4X system. Users can be one of two types: **Admin** or **Provider**.

## User Types

| Type | Description |
|---|---|
| **Admin** | Full access to manage locations, items, orders, and users |
| **Provider** | Access to their assigned location and pending orders only |

## Functions

- View all registered users
- Create a new user (Admin or Provider)
- Edit user details
- Activate / Deactivate a user account
- Assign user type (Admin / Provider)
