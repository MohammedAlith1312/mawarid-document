# Assignment

The **Assignment** page allows administrators to assign one or more **Providers** to a specific **Location**. Only assigned providers will receive orders from that location.

## Functions

- View all provider assignments for each location
- Assign a provider to a location
- Remove a provider assignment from a location

## How It Works

- A location can have **multiple providers** assigned to it.
- When a requester places an order from a seat at a location, the order is routed to the **providers assigned to that location**.
- Providers will only see orders from locations they are assigned to.
