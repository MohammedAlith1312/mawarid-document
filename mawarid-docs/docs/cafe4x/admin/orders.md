# Orders

The **Orders** page gives administrators a full history of all orders placed across all locations, with their current status.

## Functions

- View all orders across all locations
- Filter orders by location, date, status, or provider
- View order details (items ordered, seat/cabin, location, timestamp)
- Track order status

## Order Statuses

| Status | Description |
|---|---|
| **Pending** | Order placed by requester, awaiting provider action |
| **In Progress** | Provider has acknowledged and is preparing the order |
| **Completed** | Order has been delivered/fulfilled |
| **Cancelled** | Order was cancelled |
