# Agent Contract

The **Agent Contract** portal manages contract documents between Mawarid and recruitment agents, with support for **digital document signing**.

## Overview

Agent Contract is a document signing workflow application where:

- Contracts are uploaded and sent to agents for review
- Agents digitally sign the contract document
- Signed contracts are stored and tracked

## How It Works

1. Admin uploads the agent contract document
2. Document is sent to the agent for digital signature
3. Agent reviews and signs the contract digitally
4. Signed document is saved and both parties receive confirmation

## Key Features

- Contract document upload and management
- Digital signature workflow
- Signature tracking and status
- Signed document storage and download
