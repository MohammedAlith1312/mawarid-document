# Messages

The **Messages** page allows coordinators to send notifications to all employees, filtered by **Nationality** and **Language**.

## Page Type

- **Type:** Entity

## Purpose

- Compose and send broadcast messages to employees
- Target specific employee groups by nationality or language preference
- Supports multilingual notification delivery

## How It Works

1. Coordinator composes a message.
2. Selects target group by **Nationality** and/or **Language**.
3. Message is sent as a notification to all matching employees.

## Notes

- Messages can be sent in multiple languages simultaneously.
- Useful for regulatory notices, HR announcements, and project updates.
