# Request Type

The **Request Type** page is a master data entity used to define and manage the types of requests available in the Coordinator portal.

## Page Type

- **Type:** Entity (Master)

## Purpose

- Define available request categories (e.g., leave, procurement, maintenance)
- Master data that drives the Admin Requests page dropdown options
- Centralized management of request type names and codes

## Notes

- Changes to request types may affect existing requests referencing those types.
- Only admin users should modify master data.
