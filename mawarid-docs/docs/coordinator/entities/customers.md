# Customers

The **Customers** entity page is used to configure customer records that populate the top nav bar for filtering data across the Coordinator portal.

## Page Type

- **Type:** Entity

## Purpose

- Setup and manage customer entities
- Customers selected here appear in the **top nav bar** drop-down
- The top nav form loads dynamically based on app configuration

## Notes

- This is a configuration page, not a data reporting page.
- Customer records should be kept in sync with the CRM/ERP system.
