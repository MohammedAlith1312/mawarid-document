# Worker Pay Statement

The **Worker Pay Statement** page shows the payroll/pay statement for workers assigned to a project.

## Page Type

- **Type:** Remote

## API Endpoint

```
GET /getPayStatementForWorker
```

## Notes

- This endpoint is a relative path served by the portal backend.
- Pay statements are generated based on the selected project and worker.
- Details include salary, allowances, deductions, and net pay.
