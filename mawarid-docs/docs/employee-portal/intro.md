# Employee Portal

The **Mawarid Employee Portal** is a mobile-friendly self-service application that gives employees access to their personal HR and payroll information.

## Pages

| No. | Page | Description |
|---|---|---|
| 1 | [Login](./login) | Iqama-based login with OTP verification |
| 2 | [Register Mobile](./register-mobile) | Register mobile number for the employee account |
| 3 | [Update Mobile](./update-mobile) | Update existing mobile number with OTP verification |
| 4 | [Language](./language) | Switch application language |
| 5 | [My Profile](./my-profile) | View current employee details |
| 6 | [Contact Info](./contact-info) | View coordinator and manager contact details |
| 7 | [Balance](./balance) | View visa, vacation, and ticket balances |
| 8 | [Loan History](./loan-history) | View employee loan list |
| 9 | [Loan History Lines](./loan-history-lines) | View detail lines of a specific loan |
| 10 | [Payslip](./payslip) | View and download last 3 months payslips |
| 11 | [Letter Download](./letter-download) | View and download HR letters |
| 12 | [Observation](./observation) | View and create observation (complaint) records |
| 13 | [Medical Insurance](./medical-insurance) | View medical insurance details and clinic list |
| 14 | [FAQ](./faq) | View FAQ based on current language |
| 15 | [FAQ Admin](./faq-admin) | Manage FAQ content (admin only) |
