# Contact Info

The **Contact Info** page displays the current employee's **Coordinator** and **Manager** contact details.

## Functionality

- Shows coordinator and manager details linked to the logged-in employee.
- All details are returned from the Employee Details API.

## APIs

| Method | URL | Params |
|---|---|---|
| GET | `crm/GetEmployeeByID` | All details have an employee details API |
