# FAQ

The **FAQ** page shows a list of frequently asked questions filtered by the current application language.

## Functionality

- Displays FAQs relevant to the current language selected by the employee.
- Content is managed by admins via the [FAQ Admin](./faq-admin) page.

## APIs

| Method | URL | Params |
|---|---|---|
| GET | `security/faq` | `{ModuleId, Language}` |
