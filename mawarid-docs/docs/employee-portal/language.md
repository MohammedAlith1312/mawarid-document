# Language

The **Language** option is found in the top navigation bar. It allows the employee to switch the application language.

## Functionality

- Located in the **top nav bar** — uses the pipe (`|`) separator to toggle between languages.
- Language content is loaded from a local `language.json` file.
- No API call is made — the translation is handled entirely client-side.

## Data Source

| Source | Description |
|---|---|
| `language.json` | Local file containing all translations for the application |
