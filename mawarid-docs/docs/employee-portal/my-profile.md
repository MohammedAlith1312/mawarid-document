# My Profile

The **My Profile** page displays all current employee details retrieved from the CRM.

## Functionality

- Shows all employee information associated with the logged-in Iqama number.
- All details are fetched from the Employee Details API.

## APIs

| Method | URL | Params |
|---|---|---|
| GET | `crm/GetEmployeeByID` | All employee details returned from this API |
