# Survey

The **Survey** portal provides an **automatic QA (Quality Assurance) form** system for branch employees.

## Overview

The Survey application is used to automatically generate and send QA evaluation forms to branch employees. It ensures consistent quality assessment across all branches.

## How It Works

1. A QA survey form is configured and associated with branch employees
2. Forms are **automatically triggered** based on schedule or event
3. Branch employees receive and fill out the QA form
4. Responses are collected and reported

## Key Features

- Automatic QA form generation and distribution
- Branch employee targeting
- Response collection and tracking
- QA reporting per branch and employee
