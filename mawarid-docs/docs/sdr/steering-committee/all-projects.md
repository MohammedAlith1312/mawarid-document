# Steering Committee — All Projects

The **All Projects** page gives the Steering Committee a complete view of all SDR projects that passed through committee review.

## Functions

- View all projects the committee has reviewed (approved or rejected)
- Filter by status, priority, or date
- View project details including:
  - Original request summary and system
  - IT estimation and assessment
  - Assigned priority (Urgent / High / Medium / Low)
  - Current project status (Approved / In Progress / Completed / Hold / Rejected)

## Project Statuses Visible

| Status | Description |
|---|---|
| **Approved** | Committee approved; project development scheduled |
| **In Progress** | Development underway |
| **Completed** | Project fully delivered |
| **Hold** | Project paused |
| **Rejected** | Committee rejected the request |
