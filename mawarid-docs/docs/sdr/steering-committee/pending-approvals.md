# Steering Committee — Pending Approvals

The **Pending Approvals** page shows all SDR requests that have been escalated to the Steering Committee by the IT team because the estimated development effort **exceeds 21 days**.

## Functions

- View all requests pending committee decision
- View full request details:
  - Summary and selected system
  - Requirements list
  - Project benefits (entered by GM)
  - IT assessment: issues, dependencies, and estimated duration
- **Approve** the request:
  - Set project priority: **Urgent / High / Medium / Low**
  - Submit → request moves to **Approved** status and project is scheduled
- **Reject** the request:
  - Enter rejection reasons
  - Request is returned to IT or marked as **Rejected**

## Decision Flow

```
IT escalates (estimation > 21 days)
        ↓
Committee reviews request details
        ↓
   Approve?
   ├── Yes → Set Priority → Submit → Project Scheduled
   └── No  → Enter Reasons → Return to IT
```
