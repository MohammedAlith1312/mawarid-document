# Solution Development

The **Solution Development** page is the core entity of the SDR portal. It is used to create and manage software development requests.

## Page Type

- **Type:** Entity — Workflow

## Workflow Stages

`Draft` → `In Progress` → `Approved` → `Completed`

## Functions

- Create a new software development request
- Manage and track existing requests through the workflow
- All stages of the request lifecycle are handled here

## Notes

- This is the only entity in the SDR application.
- All other pages (My Requests, In Progress, Approved, Completed) are filtered views of this entity.
