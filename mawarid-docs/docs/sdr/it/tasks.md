# IT — Tasks

The **IT Tasks** page shows all development tasks associated with SDR projects. Each project is broken into multiple tasks that are assigned to individual team members.

## Views

| Tab | Description |
|---|---|
| **Pending** | Tasks not yet started or currently in progress |
| **Completed** | Tasks that have been finished |
| **All** | Full list of all tasks across all projects |

## Functions

- View all tasks with their linked project
- Create a new task under a project
- Assign a task to a team member
- Set task priority and due date
- Update task status:

| Status | Description |
|---|---|
| **Pending** | Task not yet started |
| **In Progress** | Task is being worked on |
| **Completed** | Task is done |
| **On Hold** | Task is paused |

## Notes

- Tasks are the smallest unit of work within an SDR project.
- When all tasks of a project are completed, the project status can be updated to **Completed**.
