# IT — Projects

The **IT Projects** page shows all SDR projects that have been created after requests are approved (either directly after IT review for simple efforts, or after Steering Committee approval for larger ones).

## Views

| Tab | Description |
|---|---|
| **Pending** | Projects currently in progress / scheduled |
| **Completed** | Projects that have been fully developed and delivered |
| **All** | Full list of all SDR projects |

## Project Lifecycle (SDLC)

Each project follows the **Software Development Life Cycle (SDLC)**:

```
Approved → Scheduled → In Development → Completed
```

- A project can be placed **On Hold** at any stage.
- Projects contain multiple **Tasks** managed from [IT — Tasks](./tasks).

## Project Details

Each project record includes:
- Linked SDR request
- Assigned development team members
- Priority (Urgent / High / Medium / Low) — set by Steering Committee if escalated
- Scheduled start and end dates
- Current status
- List of tasks
