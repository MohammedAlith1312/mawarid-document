# Dashboard

The **Dashboard** provides an overview of all Software Development Requests through charts and count widgets.

## Functions

- Display count widgets grouped by stage (In Progress, Approved, Completed)
- Show charts for SDR activity trends
- Quick access to filtered request views

## Widgets

| Widget | Description |
|---|---|
| My Requests Count | Total requests created by the current user |
| In Progress Count | Requests currently in the In Progress stage |
| Approved Count | Requests in the Approved stage |
| Completed Count | Requests that have been completed |
