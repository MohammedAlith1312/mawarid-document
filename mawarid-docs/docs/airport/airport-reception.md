# Airport Reception

The **Airport Reception** page displays a list of arriving flight lines managed and received by Mawarid directly.

## Page Type

- **Type:** Remote

## API Endpoint

```
GET https://portal.mawarid.com.sa/Recruitment/api/v1/recruitment/airport/getArrivingFlightLinesByMawarid
```

## Functionality

- Displays all arriving flight lines assigned to Mawarid reception
- **Action available:** Mark employee as **Arrived** or **Not Arrived**

## Displayed Information

- Flight number and arrival date/time
- Employee name and passport details
- Arrival status
