# Airport

The **Mawarid Airport Portal** is used to track and manage the arrival of employees/recruits at the airport.

## Functionality

> All pages display **list views** of flight lines and allow a single action: marking arrivals as **Arrived** or **Not Arrived**.

## Base URL

```
https://portal.mawarid.com.sa/Recruitment/api/v1/recruitment/airport/
```

## Pages

| Page | Description |
|---|---|
| [Airport Reception](./airport-reception) | Arriving flights managed by Mawarid |
| [Accommodation Reception](./accommodation-reception) | Arriving flights filtered by company (MWD) |
| [Not Arrived](./not-arrived) | Flights/employees who have not arrived |
| [Escaped](./escaped) | Escaped flight line records |
| [Receive by Customer](./receive-by-customer) | Arriving flights received directly by customer |
