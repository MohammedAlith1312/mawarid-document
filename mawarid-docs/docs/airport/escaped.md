# Escaped

The **Escaped** page displays a list of flight lines where the employee has been marked as escaped (absconded after arrival).

## Page Type

- **Type:** Remote

## API Endpoint

```
GET https://portal.mawarid.com.sa/Recruitment/api/v1/recruitment/airport/getEscapedFlightLines
```

## Functionality

- Lists all flight lines with an **Escaped** status
- **Action available:** View and track escaped employee records

## Displayed Information

- Flight number and original arrival date
- Employee name and nationality
- Date marked as escaped
- Current escalation/follow-up status

## Notes

- This page is used for compliance and HR tracking purposes.
- Escaped records may be escalated to the relevant authorities.
